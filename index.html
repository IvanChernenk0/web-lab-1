Про себе

Я вивчаю кібербезпеку та активно розвиваюся у сфері веб-розробки. Маю досвід створення архітектури проектів на JavaScript та TypeScript. Зараз фокусуюся на вивченні React та системного адміністрування.

Технічні навички

  • Frontend-розробка: Володію семантичною версткою HTML5 та сучасним CSS3. Розробка інтерактивності за допомогою JavaScript (ES6+).
  • Типізація та архітектура: Активне використання TypeScript для створення стабільного коду та розуміння принципів ООП, що допомагає в проектуванні складних сервісів.
  • Сучасні бібліотеки: Вивчаю React для побудови компонентної архітектури.
  • Інструментарій розробника: Робота з системою контролю версій Git, середовищем VS Code та терміналом.
